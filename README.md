# Roster Management System

## Overview

This project is a C++ application designed to manage and process student records. It demonstrates object-oriented programming principles, including class design, encapsulation, and data abstraction, while handling real-world data operations such as parsing and validation.

## Features

* Parses and processes structured student data from raw input
* Stores and manages student records using custom classes
* Supports adding, removing, and displaying student information
* Calculates and displays average course completion times
* Filters students by degree program
* Validates and identifies invalid email addresses

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Dynamic Memory Management
* String Parsing and Data Processing

## Key Concepts Demonstrated

* Class design and relationships (`Student`, `Roster`)
* Encapsulation and data abstraction
* Pointer usage and memory management
* Data validation and error handling
* Separation of interface (`.h`) and implementation (`.cpp`)

## Problem Solved

This project simulates a student record management system, demonstrating how structured data can be parsed, stored, and manipulated efficiently. It highlights foundational software engineering practices used in real-world applications.

## How to Run

1. Open the solution file in Visual Studio
2. Build the project
3. Run the application from `main.cpp`
